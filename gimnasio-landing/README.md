# Gimnasio Fuerza — Landing Page

Starter kit para una landing page de gimnasio.

Cómo usar:
- Abrir `index.html` en tu navegador (o usar Live Server en VS Code)
- Agregar imágenes y logo en la carpeta `assets/`
- Personalizar colores y textos en `styles.css` y `index.html`

Sugerencias de despliegue:
- GitHub Pages (push a rama `gh-pages`) o Netlify/Vercel con el repositorio.

Próximos pasos:
- Ajustes responsive y pruebas (accesibilidad y compatibilidad)
- SEO y optimización de imágenes
- Integración de formulario real o captura de leads
